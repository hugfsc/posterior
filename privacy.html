Privacy Policy

Effective date: April 2, 2026

Overview

POSTERior is a Chrome extension developed for internal use by Thermo Fisher Scientific MSD/AIG field sales teams. It helps analyze scientific conference posters using AI to generate sales intelligence briefs.

Data We Do Not Collect

POSTERior does not collect, store, or transmit any personal data to its developers. Specifically:

  • We do not collect browsing history
  • We do not collect personally identifiable information
  • We do not store poster images or analysis results on any server we control
  • We do not use cookies or tracking technologies

Data Entered by the User

Users enter the following information directly into the app:

  • Anthropic or Google Gemini API keys — stored locally in the browser only, never transmitted to our servers
  • Conference name and analyst name — stored locally in the browser, included in analysis payloads sent to the user's own Google Apps Script endpoint
  • Poster images — sent directly from the user's browser to the Anthropic API (api.anthropic.com) or Google Gemini API (generativelanguage.googleapis.com) for analysis. Images are not stored by POSTERior.

Third-Party Services

POSTERior uses the following third-party APIs, each governed by their own privacy policies:

  • Anthropic Claude API — processes poster images for analysis. See anthropic.com/privacy
  • Google Gemini API — optional alternative AI processor. See policies.google.com/privacy
  • Google Apps Script — optional logging to the user's own Google Sheets and email. Data is sent to an endpoint configured and controlled by the user.

Local Storage

Settings such as API keys, conference name, and analyst name may be saved in the browser's local storage if the user enables "Remember settings." This data remains on the user's device and is never accessed by POSTERior's developers.

Children's Privacy

POSTERior is intended for professional use by Thermo Fisher Scientific employees and partners. It is not directed at children under 13.

Changes to This Policy

We may update this privacy policy from time to time. Updates will be reflected by a new effective date at the top of this page.
